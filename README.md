# COA FSM cache controller implementation

Install [Dart SDK](https://dart.dev/get-dart) and run the following command in the terminal to execute the code:

```bash
dart main.dart
```

No external dependencies are required to run the code. The main function includes a simple test case 
to demonstrate the functionality of the FSM cache controller implementation. More requests can be added in it to further test the implementation.
Reference line 160 (apprx) in `main.dart` for the test case.
